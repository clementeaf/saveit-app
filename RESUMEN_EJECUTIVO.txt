                           CÓMO EJECUTAR
================================================================================

1. INSTALAR DEPENDENCIAS
   $ npm install

2. LEVANTAR INFRAESTRUCTURA LOCAL
   $ npm run docker:up

3. COMPILAR
   $ npm run build

4. EJECUTAR SERVICIOS (en terminales separadas)
   Terminal 1: cd services/reservation && npm run dev
   Terminal 2: cd services/qr-code && npm run dev
   Terminal 3: cd services/notification && npm run dev
   Terminal 4: cd services/channel-gateway && npm run dev
   Terminal 5: cd services/analytics && npm run dev

5. VERIFICAR HEALTH CHECKS
   curl http://localhost:3001/health
   curl http://localhost:3002/health

   curl http://localhost:3004/health
   curl http://localhost:3005/health
